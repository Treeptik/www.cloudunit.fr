Déployez votre application avec CloudUnit en moins d'1 minute

Installer une application avec git

  1. Créer une application en précisant le type de serveur
  2. Ajouter une base de données
  3. Ajouter le repository distant au projet
  4. Pousser le code sur les serveurs de CloudUnit
  5. L'application est compilée par CloudUnit puis déployée sur le serveur
  6. Enjoy ;)

Choisir CloudUnit

Le choix de l'expertise

CloudUnit est le PAAS spécialisé dans les applications Java JEE. Nous maîtrisons aussi bien les serveurs d'applications que les frameworks de développement. Nous pouvons vous accompagner dès les phases de mise en place de l'architecture, de développement jusqu'au déploiement dans le Cloud.

Le DevOps devient facile

Les méthodologies agiles sont au coeur des processus de développement d'application depuis longtemps. Il est naturel d'en attendre autant de ces infrastructures . Aujourd'hui plus question d'attendre des jours pour avoir accès à une véritable infrastructure capable d'héberger une application Java JEE. Avec CloudUnit les développeurs créent leur serveur et déploient en quelques secondes ce qui vous permet de tester et de fournir vos applications plus rapidement.

Le cloud JEE pour les devs et les admins

Développeur, déployez en poussant simplement le code source ou en utilisant l'interface graphique. Admin, intégrez la création des environnement et le déploiement de vos scripts à l'aide du client shell.

Ne jetez pas vos serveurs

CloudUnit s'installe aussi bien sur vos machines physiques que virtuelles et nous vous accompagnons dans la mise en place d'un cloud privé.
